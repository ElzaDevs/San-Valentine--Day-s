## 📖 Overview

SanValentine is a static front-end project designed to create a memorable digital Valentine's Day experience.

The website combines:

* 💌 Personalized love messages
* 🎵 Integrated music playlist
* 🌸 Animated decorative elements
* ✨ Y2K-inspired aesthetics
* 💖 Interactive navigation experience

The project runs entirely in the browser without requiring any backend infrastructure.

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/elzadevs/San-Valentine--Day-s.git
```

Open the project:

```bash
cd San-Valentine--Day-s
```

Launch:

```text
index.html
```

Or use VS Code Live Server.
